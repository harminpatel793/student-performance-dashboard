# 🎓 Student Performance Analysis Dashboard

Real-world analysis of 6606 students using Kaggle dataset.

## 🛠️ Tech Stack
- Python
- Pandas
- Plotly
- Streamlit
- Kaggle Dataset

## ✨ Features
- Analyze 6606 real students
- Filter by Gender, School Type, Motivation
- Score range slider
- Score distribution charts
- Correlation analysis
- Key insights section
- Download filtered data

## 🔍 Key Findings
- Attendance is #1 factor (0.58 correlation)
- High motivation = better scores
- Sleep barely affects scores 😂
- Gender has almost zero impact

## 🚀 How to Run
pip install streamlit plotly pandas kagglehub
streamlit run student_analysis.py
